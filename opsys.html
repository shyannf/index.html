Operating Systems Terminology
Term Definition
Operating System The software that controls the overall operation of a computer including file management, the user interface, and the program execution environment.
Job Queue In batch processing systems, the jobs residing in mass storage wait for execution in a Job Queue.
FIFO A first-in,first-out fashion in a storage organization in a queue.
Interactive Processing A feature that allowed a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing The solution for the operating systems that helps the to be able to provide service to multiple users at the same time.
Multiprogramming This is when time is divided into intervals and then the execution of each job is restricted to only one interval at a time, at the end of each interval the job is temporarily set aside and another is put in place, this makes it to where the jobs can be shuffled one by one making is easier to do multiple intervals at a time.
Application Software The programs for performing tasks particular to the machine's utilization.Some examples would be a spreadsheet or a desktop publishing system.
System Software Performs the tasks that are commoon to computer systems in general. Provides the infrastructure that the application software requires.
Utility Software Programs for performing activities that are fundemental to computer installations but its not included in the operating system.
Graphical User Interface (GUI) When files and programs are manipulated, they are represented pictorially on the display as icons.
Shell The older user interfaces from GUI, shells communicated with users through textual images using a keyboard and moniter screen.
Kernel The internal part od an operating system, contains software components that perform the very basic functions required by computer installation.
Device Driver A component of the kernel, software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager A unit of the kernel, coordinate the use of the machine's mass storage facilities.
Memory Manager A component of the kernel, charged with the task of coordinating the machine's use of main memory.
Virtual Memory A large "fictional" memory space created by paging.
Boot Strapping Performed on a computer, this is an operating system that provides the software infastructure required by other software units.
ROM A small portion on a computers main memory where the CPU expects ro find its initial program is constructed from special nonvolative memory cells.
Deadlock The condition that two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time.
Super User (or administrator) This person has highly privileged access to the operating system.
Multi-Factor Authentication These are more techniquilly it is another password that helps verify that it is you and not someone else logging into your acoount.
